# ✈️ Airline Agent (Python Project)

An AI-inspired Airline Agent built using Python that allows users to search and book flights through an interactive interface. This project simulates how real-world flight booking systems work by integrating an agent-based architecture with a custom-built flight API.

---

## 🚀 Features

* 🔍 Search flights between cities
* 🧠 Agent-based logic for handling user queries
* 🔗 Simulated Flight API (mimics real-world APIs like Amadeus & Skyscanner)
* 🎟️ Flight booking functionality
* 🖥️ Interactive GUI built with Tkinter
* 🎨 Modern and user-friendly interface

---

## 🧩 Project Structure

```
airline-agent/
│
├── app.py           # GUI application
├── main.py          # CLI version (terminal-based)
├── agent.py         # Core agent logic
├── flight_api.py    # Simulated flight API
├── booking.py       # Booking functionality
```

---

## ⚙️ How It Works

This project follows an agent-based architecture:

User Input → Agent → API → Results → Booking

* The user enters source and destination
* The agent processes the request
* The system fetches data from the simulated API
* Available flights are displayed
* The user selects and books a flight

```

```

```

3. Run the GUI application:

```
