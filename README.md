✈️ Built My Own AI-Powered Airline Agent (Python Project)

Excited to share my latest project — an Airline Agent that allows users to search and book flights through an interactive system.

🔹 Key Features:
• Search flights between cities
• Simulated Flight API (real-world API concept)
• Flight booking functionality
• Interactive GUI built with Tkinter
• Clean and user-friendly design

🔹 Technical Approach:
To simulate real-world systems, I designed a custom flight API that mimics platforms like Amadeus and Skyscanner.
The project follows an agent-based architecture where the system processes user input, interacts with a data source (API), and returns structured results.

🔹 What I Learned:
• How APIs work in real applications
• Basics of AI/Agent-based systems
• GUI development using Python (Tkinter)
• Problem-solving and debugging skills
• Building real-world projects with zero budget

💡 This project was inspired by concepts from the OpenAI Agent SDK, focusing on how intelligent systems interact with tools and user queries.
